About This Project

This game was made as the final project of the Frontendmasters' frontend bootcamp.
It uses only HTML, CSS and Javascript.

I uploaded it just for fun, and to share it with other people, for non-profit purposes.

The game was built for PC, so I would recommend it to get the full experience. However, you can also play the game on your mobile phone.
